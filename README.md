# ArchitectTasks

A Swift package for architect task management.

## Usage

Add this package to your dependencies in `Package.swift`:

```
.package(url: "<your-repo-url>", from: "0.1.0")
```

## Structure
- Sources/ArchitectTasks/ArchitectTasks.swift: Main module
- Tests/ArchitectTasksTests/ArchitectTasksTests.swift: Unit tests

## Build & Test

```
swift build
swift test
```
